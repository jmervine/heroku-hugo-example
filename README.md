# Quick Start

A basic example of a [Hugo] generated site running on [Heroku] using a simple custom [Go] static file server.

#### Requirements

- [Go]
- [Hugo]
- [Godep]
- [Heroku] Account

[Go]: https://golang.org/
[Hugo]: https://gohugo.io/
[Godep]: https://github.com/tools/godep
[Heroku]: https://www.heroku.com/
